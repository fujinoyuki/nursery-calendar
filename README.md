# 保育園カレンダーアプリケーション

## 概要
このアプリケーションは、保育園の年間行事を管理するためのカレンダーアプリケーションです。

## 機能
1. イベントの一覧表示
2. イベントの追加・編集・削除
3. ユーザー認証
4. Supabaseとの連携

## 技術スタック
- Next.js
- TypeScript
- Supabase
- Tailwind CSS

## セットアップ
1. リポジトリをクローン
```bash
git clone [repository-url]
```

2. 依存関係のインストール
```bash
npm install
```

3. 環境変数の設定
```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
```

4. 開発サーバーの起動
```bash
npm run dev
```
