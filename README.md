# 保育園カレンダー

保育園の年間行事を管理するためのアプリケーションです。

## 機能

- 月別のイベント管理
- イベントの追加・編集・削除
- 年齢グループ別のイベント表示
- イベントの詳細情報（準備物、目的など）の管理

## 技術スタック

- Next.js
- TypeScript
- Supabase
- Tailwind CSS

## セットアップ

1. リポジトリをクローン

```bash
git clone https://github.com/yourusername/nursery-calendar.git
```

2. 依存関係のインストール

```bash
npm install
```

3. 環境変数の設定

```bash
cp .env.example .env.local
```

4. 開発サーバーの起動

```bash
npm run dev
```
